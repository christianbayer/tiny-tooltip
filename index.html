Options

Option name Description Default value
delay Delay for the tooltip appear (in milliseconds) 0
class Custom class for the tooltip element
removeOnScroll If the window scrolls, the tooltip disapear true
rem Default value for 1 rem $("body").css('font-size')
disableDelayOnTouch If there will be delay on touch devices true
